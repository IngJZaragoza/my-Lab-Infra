# Fundamentos de Seguridad de la Infraestructura del laboratorio (FASE 1)

## Objetivo del Proyecto
Desplegar los cimientos de un entorno de servidores de nivel empresarial, enfocado en la alta disponibilidad desde el almacenamiento (LVM), la seguridad de acceso (SSH Hardening) y la trazabilidad del cambio mediante estándares de industria (Conventional Commits).

Este laboratorio representa la base técnica para una infraestructura escalable, priorizando la seguridad sobre la facilidad de uso.

## Stack Técnico
* **Virtualización:** VMware.
* **SO:** Ubuntu Server 24.04 LTS.
* **Almacenamiento:** Gestión de volúmenes lógicos (LVM) para escalabilidad caliente.
* **Seguridad:** Acceso remoto exclusivo mediante llaves criptográficas y Hardening de SSH.
* **Control de Versiones:** Gestión de configuración mediante Git bajo el estándar de Conventional Commits.
